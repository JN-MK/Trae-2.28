# AI Life Coach 项目说明

## 项目简介
这是一个基于 DeepSeek R1 API (火山方舟) 的 AI Life Coach 网页应用。通过与 AI 对话，用户可以获得生活建议和成长指导。本项目包含一个简单的前端界面和一个 Node.js 后端服务器，用于处理 API 请求并解决跨域问题。

## 项目结构
- `index.html`: 网页的主入口文件，包含聊天界面的 HTML 结构。
- `style.css`: 网页的样式文件，负责页面的布局和美化，采用响应式设计。
- `script.js`: 网页的逻辑文件，负责处理用户输入、发送请求到后端并显示回复。
- `server.js`: Node.js 后端服务器，用于转发请求到 DeepSeek API，保护 API Key 并处理 CORS。
- `package.json`: 项目依赖配置文件。

## 如何运行
1. 确保已安装 Node.js。
2. 在项目根目录下打开终端。
3. 运行 `npm install` 安装依赖。
4. 运行 `node server.js` 启动服务器。
5. 在浏览器中打开 `http://localhost:3001` (或直接打开 index.html，但建议通过服务器访问)。
